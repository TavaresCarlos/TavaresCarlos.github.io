TerrainIntelGIS

Sistema que utiliza somente tecnologias openssource, voltado para o apoio à tomada de decisão em áreas de desastres ambientais. O sistema se baseia no upload de um arquivo csv pelo usuário. A partir disso, diversos algoritmos de Machine Learning são executados com a finalidade de gerar agrupamentos baseados em dados ocultos do arquivo do usuário. A partir disso, é possível visualizar as informações.
